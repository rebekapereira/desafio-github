Comandos básicos: (windows)  

 

Dir = traz uma lista de diretórios contidas na pasta a qual estou situada  

 

Cd /   = permiti que  a gente navegue entre as pastas 

 

Cd .. = volta uma navegação / sai da pasta 

 

Ctrl + l =  limpa a tela    

 

TAB = quando usamos a tecla TAB ele completa o nome da pasta que está dentro do windows 

 

Mkdir = para criar pastas exemplo: mkdir workspace 

 

Echo = Para criar um arquivo = exemplo: 

Echo hello > hello.txt 

 

Nos dois exemplos acima foi criado uma pasta chamada workspace e um arquivo dentro dessa pasta chamado hello.txt 

 

Del = caso eu queira excluir tudo que estiver dentro de tal pasta, exemplo:  

Del workspace, ira deletar todos arquivos dentro da pasta workspace 

 

Rmdir = para deletar/remover repositório e tudo que estiver dentro, exemplo: 

Rmdir workspace /S  / Q

Iniciando o Git e criando um commit:

Git add = mover arquivos e dar início ao versionamento 

Git commit = para criar o commciit
